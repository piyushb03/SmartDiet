#  SmartDiet 💪🏼

SmartDiet is an AI-powered nutrition tracking application that helps users monitor daily macros, log meals, and generate personalized diet plans based on their goals and biometrics.

---

## 🚀 Project Structure

This repository is divided into two main parts:

* 📦 **Client** → Frontend built with React + Vite
* ⚙️ **Server** → Backend API with AI meal generation

```
SmartDiet/
├── client/   # Frontend (React + Vite)
├── server/   # Backend (Node.js / API / AI integration)
```

---

## ✨ Features

* 📊 Macro tracking dashboard
* 🤖 AI-generated meal plans
* 📔 Food diary logging system
* 🔐 Secure authentication
* 🌗 Dark/light mode UI

---

## 🛠️ Tech Stack

* **Frontend:** React, Vite, Tailwind CSS
* **Backend:** Node.js, Express
* **Auth & DB:** Supabase
* **AI:** Gemini / Groq

---

## ⚡ Quick Start

Clone the repository:

```bash
git clone https://github.com/piyushb03/SmartDiet.git
cd SmartDiet
```

### Run Client

```bash
cd client
npm install
npm run dev
```

### Run Server

```bash
cd server
npm install
npm start
```
